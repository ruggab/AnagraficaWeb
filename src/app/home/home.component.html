Generazione Hash Anagrafica Vero

Quest'applicazione permette di leggere un file csv con uno specifico formato e di generare il codice HASH per ogni record del file.

Di seguito il fomrato che dovrà avere il file csv:

Intestazione Colonne: tessera; data fine validita, cognome; nome; sesso; data; nazione; provinca; comune

Separatore: Catattere 'virgola' (;)

Tutti i campi saranno di tipo testo e la data (data nascita) in nparticolare dovrà avere il seguente: gg-mm-aaaa
